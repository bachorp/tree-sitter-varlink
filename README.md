# tree-sitter-varlink

[![crates][crates]](https://crates.io/crates/tree-sitter-varlink)

[Tree-sitter](https://tree-sitter.github.io/) support for [Varlink](https://varlink.org/).
Includes queries for [highlights](./queries/highlights.scm) and [tags](./queries/tags.scm).
Provides bindings for [Rust](https://rust-lang.org/).

[crates]: https://img.shields.io/crates/v/tree-sitter-varlink?logo=rust
